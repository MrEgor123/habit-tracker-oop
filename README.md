# Менеджер привычек (лабораторная работа №4)

Небольшая консольная программа на Python для отслеживания привычек, разработанная в рамках лабораторной работы №4
по теме "Коллективная разработка с использованием парадигмы ООП. Code Review".

## Цели проекта

- Закрепить навыки объектно-ориентированного программирования (ООП);
- Отработать командную разработку с использованием Git и GitHub;
- Научиться проводить Code Review и оформлять результаты.

## Основной функционал

- добавление привычки;
- просмотр списка привычек;
- отметка выполнения привычки;
- просмотр количества выполнений по каждой привычке.
